<h1>NeuroFix</h1>

  <p>
    <strong>NeuroFix</strong> is a responsive mental wellness web application that helps users
    track moods, journal thoughts, and practice mindfulness through simple self-care tools.
    The app focuses on calm design, accessibility, and emotional well-being.
  </p>

  <hr />

  <h2>Key Features</h2>
  <ul>
    <li>Mood tracking with interactive calendar</li>
    <li>Personal journaling space</li>
    <li>Meditation timer for mindfulness</li>
    <li>Clean feedback form</li>
    <li>Fully responsive, calm UI</li>
  </ul>
